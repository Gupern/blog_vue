# 命令一览

### 安装vue cli
`npm install -g @vue/cli @vue/cli-service-global` 

### 创建vue项目
`vue create hello-world`

## Project setup
`npm install`

### Compiles and hot-reloads for development
`npm run serve`

### Compiles and minifies for production
`npm run build`

### Lints and fixes files
`npm run lint`
